## Information
The Toy Robot Exam project was developed using Typescript. The application executes each command on each user input line. Unit Test cases are implemented in test folders.

<b>Prerequisites</b>:
Please install packages before running the project
```
npm install
```

## Running the project
You can run the project using 2 options provided:
1. Running the console app on index.ts:
```
npm run start
```
2. Running Jest tests:
```
npm run test
```
or to run test coverage
```
npm run coverage
```
